============
Installation
============

Installing contagion
--------------------

Install ^^contagion^ with pip_:

::

    pip install contagion


Upgrading contagion
--------------------

Upgrade ^^contagion^ with pip_:

::

    pip install contagion --upgrade


Requirements
------------

This project was created with:

- matplotlib_==3.2.1
- numpy_==1.16.6
- networkx_==2.4
- seaborn_==0.8.1
